Description:

  1.This C program checks whether a given number is divisible by 9.
  
  2.If the number is divisible by 9, the program prints 9.
  
  3.Otherwise, it prints the remainder when the number is divided by 9.


        
Concepts Used:


1.Variables

2.scanf() for user input

3.Modulus (%) operator

4.Ternary (?:) operator

5.printf() for output
